# Usage
FindCMS takes args like

- -u (--url) – Website on which the CMS needs to be detected
- -f (--file) – List of domains/URLs on which the CMS needs to be detected
- -v (--verbose) – Detailed scan information (what exactly is being checked at the moment)
- -o (--output) – File where detected CMS results are saved, in the format Website|CMS
- -c (--cms) – CMS that has priority (the scanner will check only this CMS)
- -t (--threads) – Number of threads used for scanning
